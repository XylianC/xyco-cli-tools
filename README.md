# XCU Command Line Tools

This is the start of a series of programs I write to make my own terminal utils.
I chose the name XCU as a sort of joke XC are my initials and CU means core utils


Specifically the goal is to learn how popular commandline commands work by duplicating their behaviour in c.

When I started this project I thought I'll just quickly learn these things, but then taking some time to think.
I understood what a great learning opportunity this project could be, if taken seriously.

As a proof of concept I have added a small demo: 
![GIF Demonstrating custom echo application](https://raw.githubusercontent.com/XylianC/xyco-cli-tools/master/documentation/gifs/echo-demo-tool.gif)

Here are some topics I think I will learn:
## Programming
* C 	- 	programming language
* C 	- 	pointers, memory allocation, efficient code
* C 	- 	C is everywhere, what is it used for?
* Makefiles - Compiling binaries

## Linux
* Actually understand how command line tools I use a lot work:
	- ls
	- man
	- pwd
	- cd
* Kernel Behaviour
* General Linux skills: terminal workspace, man pages, regex

## Git
* How to efficiently use git
* gh (same but from terminal)
* branches
* documentation

## Furthermore
I genuinly think my all-round linux understanding, operating system behaviour (like the kernel) and my skills as a programmer will be properly tested.
I hope to learn to use linux manpages efficiently, linux kernel behaviour, the c programming language, and maybe even... vim! :)

* git - version control
